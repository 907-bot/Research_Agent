## Troubleshooting

### Common Issues:

1. **Build fails with package errors**
   - Check requirements.txt formatting
   - Ensure no typos in package names

2. **Model download timeout**
   - The sentence-transformers model downloads on first run
   - This is normal and happens only once
   - Give it 2-3 minutes

3. **Search not working**
   - DuckDuckGo might rate-limit requests
   - Wait a few seconds between searches

4. **Memory errors**
   - Upgrade to a better hardware tier (free tier should work though)
   - Consider using a smaller embedding model

## Performance Tips

- First search is slower (model loading)
- Subsequent searches are faster
- Average query time: 10-30 seconds
- Works best with specific queries
